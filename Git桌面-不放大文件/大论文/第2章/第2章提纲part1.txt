## 第 2 章 基础理论介绍（示例提纲）

### 2.1 联邦学习概述
1. **联邦学习的背景与动机**  
   - 数据孤岛与隐私保护需求  
   - 集中式学习的缺陷与联邦学习的优势  
2. **联邦学习的定义**  
   - 基本思想：在不交换原始数据的前提下进行协作建模  
   - 常用的联邦学习范式：横向（HFL）、纵向（VFL）与联邦迁移学习等  
3. **横向与纵向联邦学习区别**  
   - **横向联邦学习**（HFL）：特征空间相同、样本分布不同  
   - **纵向联邦学习**（VFL）：样本 ID 空间相同、特征空间互不重叠（第 3 章聚焦的主要场景）  
4. **联邦学习的安全与隐私保护机制**  
   - 常见的安全多方计算（Secure Multi-Party Computation）  
   - 同态加密、差分隐私等方法在联邦学习中的应用  
   - 私有集交集（PSI）协议在多方样本对齐中的作用  

### 2.2 半监督学习基础
1. **半监督学习定义**  
   - 介于有监督学习和无监督学习之间  
   - 充分利用少量标注数据和大量未标注数据  
2. **常见的半监督学习方法**  
   - 自训练、共训练、伪标签、图半监督等  
   - 半监督学习在推荐系统、医疗诊断等领域的典型应用与挑战  
3. **PU（Positive and Unlabeled）学习**  
   - 仅有正样本与未标注样本的特殊半监督场景  
   - 关键难点：未标注数据中既包含真正的负样本也可能包含正样本  
   - 常见方法：两步技术、PU Bagging 等  

### 2.3 纵向联邦场景下的半监督与 PU 学习
1. **纵向联邦与半监督学习的结合动机**  
   - 多机构/多企业间数据存在特征互补且无法直接共享  
   - 标注不足或仅有单侧机构拥有部分正样本的现实问题  
2. **未标记数据缺失的 PU 学习（UDD-PU）问题**  
   - UDD-PU 的提出：当正样本与未标注样本分属不同数据拥有方  
   - 与传统 PU 学习的主要区别与挑战  
   - 该问题在现实金融、医疗及推荐系统等场景中的典型应用需求  

### 2.4 加密与安全通信技术
1. **私有集交集（Private Set Intersection, PSI）**  
   - 用途：安全对齐多方的重叠样本 ID  
   - 常见协议：基于盲 RSA、Diffie-Hellman 或同态加密的 PSI  
2. **同态加密（Homomorphic Encryption）**  
   - Paillier 加密在联邦梯度传输、损失计算等环节的应用  
   - 同态加密对系统通信与计算的额外开销  
3. **安全多方计算（Secure Multi-Party Computation）**  
   - 基本思想与原理  
   - 在纵向联邦训练中用于保护梯度、损失等中间结果  

### 2.5 常用机器学习算法在联邦场景下的适配
1. **线性模型：逻辑回归（LR）**  
   - 联邦逻辑回归的梯度交换与加密机制  
   - 优点：易实现、计算量小、可解释性强  
2. **基于树的模型**  
   - 决策树、随机森林（RF）、梯度提升决策树（GBDT）、XGBoost、LightGBM 等  
   - 在联邦场景下的核心改造（安全分裂节点、加密梯度与损失等）  
   - 适用场景与优势：对特征分布敏感度低、可处理非线性、训练效率高  
3. **PU Bagging 在联邦场景下的启示**  
   - 通过对未标记样本进行多轮采样、训练与袋外估计（OOB）来识别可靠正样本  
   - 在分布式环境下对样本的抽样、整合与投票机制  

### 2.6 常见评价指标
1. **二分类常用指标**  
   - 准确率（Accuracy）、精确率（Precision）、召回率（Recall）、F1-score  
   - ROC 曲线和 AUC  
2. **PU 学习中的指标考量**  
   - 未标记数据中正负分布不明时，精确率-召回率曲线的重要性  
   - 评价“可靠正样本”识别效果的特殊指标  

### 2.7 本章小结
1. **总结**  
   - 本章围绕联邦学习、半监督学习及 PU 学习的核心理论进行了介绍  
   - 重点讨论了纵向联邦框架下隐私保护、对齐技术及典型算法的适配  
   - 为第 3 章所提出的“基于多方联邦的半监督学习方法”奠定了理论基础
